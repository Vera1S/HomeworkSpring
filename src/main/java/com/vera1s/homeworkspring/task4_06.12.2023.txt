 Создайте собственную аннотацию, которая будет видна во время runtime и разрешено ее
 использовать только для полей и классов. У аннотации должно быть одно поле - текстовое
  с названием value.

Второй уровень домашки: создайте аннотацию, которая будет ставиться на классе.


В Main сделайте такую реализацию: в классах, на которых указана аннотация, все ссылочные
 поля заполняются созданными вами (через рефлексию) объектами с использованием конструкторов
  по умолчанию.
